# Minecraft 1.21.10 Client For Skyblock

编程初学，这个客户端大概是在 **2025 年 8 月** 开始制作的。原本打算自己做一些功能改善自己的 **1.21.5** 游玩体验，后面逐渐完善了很多功能，到 **9月22号** 基本上完成到现在的程度

本来想要做的较为完整再公开，但是因为学习原因不得不退游 + 终止开发。最近（**2026.02**）发现游戏主流版本已经来到 **1.21.10**，因为我实在没有时间与更多兴趣去完善了，所以在进行约 4 个小时的版本同步后，把客户端直接上传到 Github

**注意说明：**

* 本客户端因为个人水平原因和最初的开发原因，大量使用 **AI** 与参考了 **raven-bs** 客户端的框架，以及 **Meteor** 在高版本的方法。因此同样采用 **GNU General Public License v3.0** 开源协议
* 代码可能存在大量冗余和拖累进程的地方，介意可以不使用
* 如果后续使用游戏出现启动崩溃，可以删除 **config/Frosty** 文件夹
* 我大概率不会在短期内或永远不会继续制作，如果有人愿意优化，感激不尽

截至当前，客户端仅留存一些针对 **Foraging Update** 最初的部分功能，以及一些基本的 **Skyblock** 功能

* **Foraging Macro**：因为很多原因，**不会**继续开发，且**不会**发布已有源码
